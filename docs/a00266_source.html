47// for example, for ISEC_CT over current protection
48// if fault clear:
49// if ISEC_CT > ISEC_OC_THRES_TRIG_AMPS for longer than ISEC_OC_T_BLANK_TRIG_SEC
50// then set fault
51// if fault set:
52// if ISEC_CT <= ISEC_OC_THRES_CLEAR_AMPS for longer than ISEC_OC_T_BLANK_CLEAR_SEC
53// then clear fault
54// ISEC_OC_TICK_SEC is the rate at which the fault handler is invoked
