52// for example, for ISEC_CT over current protection
53// if fault clear:
54// if ISEC_CT > ISEC_OC_THRES_TRIG_AMPS for longer than ISEC_OC_T_BLANK_TRIG_SEC
55// then set fault
56// if fault set:
57// if ISEC_CT <= ISEC_OC_THRES_CLEAR_AMPS for longer than ISEC_OC_T_BLANK_CLEAR_SEC
58// then clear fault
59// ISEC_OC_TICK_SEC is the rate at which the fault handler is invoked
