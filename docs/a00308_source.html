58
59// all fault parameters follow the same convention
60// for example, for ISEC_CT over current protection
61// if fault clear:
62// if ISEC_CT > ISEC_OC_THRES_TRIG_AMPS for longer than ISEC_OC_T_BLANK_TRIG_SEC
63// then set fault
64// if fault set:
65// if ISEC_CT <= ISEC_OC_THRES_CLEAR_AMPS for longer than ISEC_OC_T_BLANK_CLEAR_SEC
66// then clear fault
67// ISEC_OC_TICK_SEC is the rate at which the fault handler is invoked
68// it is required for the firmware to measure trigger and clear blanking times
