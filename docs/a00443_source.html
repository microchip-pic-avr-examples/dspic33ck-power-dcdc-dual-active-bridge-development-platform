82// // cannot use LEB feature in the dsPIC33C as it is, because LEB triggering

85// // use duty of this PWM as the LEB signal

91// // tie CLC2 output to a virtual pin RP176
92// // tie RP176 to PCI input 9
93// // then use PCI input 9 as trigger for PWM6 (the phantom PWM)
94// // this means that PWM6 will run every time PWM1H changes state
95// // during the duty cycle of PWM6, comparator faults on all PWMs are ignored, as PWM6 is used as
96// // the PCI fault acceptance qualifier for all PWMs
